# 👕 Fashion MNIST Classification using CNN

This project demonstrates image classification using a Convolutional Neural Network (CNN) built with TensorFlow and Keras. The model is trained on the Fashion MNIST dataset to learn visual features from grayscale images of clothing items. The notebook covers the complete deep learning workflow, including data preprocessing, image normalization, CNN model construction, training, evaluation, and saving the trained model architecture and weights for future use.

## ✨ Features
- Data loading and preprocessing
- Image normalization
- CNN architecture with convolution and max-pooling layers
- Dropout for regularization
- Model training and evaluation
- Saving the trained model architecture and weights

## 🛠️ Tech Stack
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib
- Jupyter Notebook

## 📂 Dataset
The project uses the **Fashion MNIST** dataset, consisting of **70,000 grayscale images (28×28 pixels)** across **10 clothing categories**, including T-shirts, trousers, dresses, coats, sandals, sneakers, bags, and ankle boots.

## 📌 Workflow
Dataset Loading → Image Preprocessing → CNN Model Building → Training → Evaluation → Save Model

## 📁 Output
The trained model is saved as:
- `cnn_mnist_model_structure.json` – Model architecture
- `cnn_mnist_model_weights.h5` – Trained model weights

## 🎯 Learning Outcomes
- Understanding CNN architecture for image classification
- Image preprocessing and normalization
- Model training and evaluation using TensorFlow/Keras
- Saving trained models for future inference or deployment
